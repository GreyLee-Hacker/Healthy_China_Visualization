# 健康中国建设追踪评估可视化项目

## 一、模块设计

1. **基础数据**：展示数据整体情况
2. **评估指标**：展示国家和省级层面、部门和行动层面评价指标
3. **中央维度**：展示国家层面年度评估结果（时间趋势）
4. **省级维度**：展示31省市评估结果
5. **部门维度**：展示部门评估结果
6. **行动维度**：展示15项专项重点行动评估结果
7. **部门协同网络**：政策网络（需计算邻接矩阵）
8. **政策树**：展示政策沿袭（需计算政策演化情况）

### （三）中央维度
见配套数据“中央层面评估结果”，可以以年份为横坐标，各级指标得分为纵坐标画折线图。

### （四）省级维度
见配套数据“省级层面评估结果”，可用全国标准地图，颜色深浅表示得分高低，既可以展示时间变动趋势，又可以展示不同层次指标得分。

### （五）部门维度
见配套数据“部门维度评估结果”，可用条形图或者直方图展示得分最高的10个部门（得分由高到低排列，分层次分时间展示）。

### （六）行动维度
见配套数据“行动层面评估结果”，可用条形图或者直方图展示各项行动的得分（得分由高到低排列，分层次分时间展示）。

### （七）部门协同网络
需要借助配套数据“国家层面政策数据源”和“省级层面政策数据源”中的部门标准化以及年份、行动、省份等信息，计算出参与部门的共现矩阵并分不同属性（国家/省级；年份；15项行动）展示。

### （八）政策树
需要借助配套数据“国家层面政策数据源”和“省级层面政策数据源”中的政策标题以及政策沿袭，结合时间、行动、省份等信息，绘制出政策沿袭的树状图并分不同属性（国家/省级；年份；15项行动）展示。

## 三、可添加的模块

### （一）主题词频
需要依靠大模型从政策文本中提取关键词，借助配套数据“国家层面政策数据源”和“省级层面政策数据源”中的政策标题以及政策文本，统计汇总主题热词（前10名），然后分年份、行动或省份显示。

### （二）政策主题以及其他发现
这里也可以介绍三大健康领域的政策占比趋势以及中观服务层面的评价。以上内容均为政策数据，这里也可以提到部分统计年鉴数据的评价结果，包括健康服务与保障、健康影响因素控制、重点人群健康促进3个层面。

- 国家层面2016-2023年各健康领域发文量占比
- 省级层面2016-2023年各健康领域发文量占比

## 项目设置

### 安装依赖

在开始之前，请确保你已经安装了Node.js和npm。然后运行以下命令来安装项目的依赖：

```sh
npm install
```

### 开发环境

运行以下命令来启动开发服务器：

```sh
npm run serve
```

开发服务器启动后，打开浏览器并访问http://localhost:8080即可查看项目。

### 生产环境

运行以下命令来编译项目以供生产环境使用：

```sh
npm run build
```

编译后的文件将会生成在dist目录下。

### 代码检查

运行以下命令来检查和修复代码：

```sh
npm run lint
```

### 代码结构

`src/data.js`
该文件包含了示例政策数据：

```js
export const policies = [
  { id: 1, name: '政策A', description: '这是政策A的描述', date: '2023-01-01' },
  { id: 2, name: '政策B', description: '这是政策B的描述', date: '2023-02-01' },
  { id: 3, name: '政策C', description: '这是政策C的描述', date: '2023-03-01' }
];
```

`src/components/PolicyTable.vue`
该组件用于展示政策数据：

```vue
<template>
  <div>
    <h2>国家政策数据</h2>
    <table>
      <thead>
        <tr>
          <th>ID</th>
          <th>名称</th>
          <th>描述</th>
          <th>日期</th>
        </tr>
      </thead>
      <tbody>
        <tr v-for="policy in policies" :key="policy.id">
          <td>{{ policy.id }}</td>
          <td>{{ policy.name }}</td>
          <td>{{ policy.description }}</td>
          <td>{{ policy.date }}</td>
        </tr>
      </tbody>
    </table>
  </div>
</template>

<script>
import { policies } from '../data';

export default {
  name: 'PolicyTable',
  data() {
    return {
      policies
    };
  }
};
</script>

<style scoped>
table {
  width: 100%;
  border-collapse: collapse;
}

th, td {
  border: 1px solid #ddd;
  padding: 8px;
}

th {
  background-color: #f2f2f2;
}
</style>
```

`src/App.vue`
主应用文件，引入并使用`PolicyTable`组件：

